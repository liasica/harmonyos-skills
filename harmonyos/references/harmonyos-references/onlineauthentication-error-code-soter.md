## 1014500001 服务异常

PhonePC/2in1Tablet

**错误信息**

The service is abnormal.

**错误描述**

SOTER服务异常时，系统会产生此错误码。

**可能原因**

SOTER服务异常。

**处理步骤**

1. 检查网络状况。
2. 重启系统。
