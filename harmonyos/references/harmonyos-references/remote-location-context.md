RemoteLocationExtensionContext是[RemoteLocationExtensionAbility](remote-location-ability.md)的上下文环境，继承自[ExtensionContext](js-apis-inner-application-extensioncontext.md)。

**模型约束：** 属性仅可在Stage模型下使用。

**系统能力：** SystemCapability.Push.PushService

**起始版本：** 4.1.0(11)

## 导入模块

PhonePC/2in1TabletTVWearable

```
1. import { RemoteLocationExtensionContext } from '@kit.PushKit';
```

## RemoteLocationExtensionContext

PhonePC/2in1TabletTVWearable

**模型约束：** 属性仅可在Stage模型下使用。

**系统能力：** SystemCapability.Push.PushService

**设备行为差异：** 该属性在Phone、PC/2in1、Tablet中可正常使用，在其他设备类型中无效果。

**起始版本：** 4.1.0(11)

本类继承自[ExtensionContext](js-apis-inner-application-extensioncontext.md)，未新增内容。
