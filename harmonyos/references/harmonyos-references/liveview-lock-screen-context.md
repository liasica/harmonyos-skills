## 导入模块

PhonePC/2in1Tablet

```
1. import { LiveViewLockScreenExtensionContext } from '@kit.LiveViewKit';
```

**设备行为差异：** 该模块在Phone、Tablet中可正常调用，在其他设备类型中无效果。
