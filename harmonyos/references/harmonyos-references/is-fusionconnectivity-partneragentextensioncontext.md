PartnerAgentExtensionContext模块是三方外设的发现和连接管理功能的上下文。

说明

* 本模块接口从API version 23开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。
* 仅可在Stage模型下使用。

## 导入模块

PhonePC/2in1Tablet

```
1. import { PartnerAgentExtensionContext } from '@kit.ConnectivityKit';
```

## PartnerAgentExtensionContext

PhonePC/2in1Tablet

PartnerAgentExtensionContext模块是三方外设的发现和连接管理功能的上下文。继承自[ExtensionContext](js-apis-inner-application-extensioncontext.md)类，提供了PartnerAgentExtensionAbility的相关上下文。

**系统能力**：SystemCapability.Communication.FusionConnectivity.Core

**模型约束**： 此接口仅可在Stage模型下使用。
