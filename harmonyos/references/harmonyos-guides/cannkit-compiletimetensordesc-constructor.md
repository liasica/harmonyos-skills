## 函数功能

CompileTimeTensorDesc类用于描述编译时的Tensor描述信息，包含dtype信息以及format信息。由于编译时无法确定shape，因此不包含shape信息。该函数为CompileTimeTensorDesc类的构造函数。

## 函数原型

```
1. CompileTimeTensorDesc()
```

## 参数说明

无

## 返回值

返回一个CompileTimeTensorDesc对象。

无
