CodeGenie支持在**对话框中**输入对代码段和代码文件分析要求，帮助开发者快速理解代码逻辑、代码功能、技术细节和潜在问题等，提升开发效率。

DevEco Studio 6.0.2 Beta1之前版本，分析代码文件时需要通过**Files**入口选中文件；分析代码片段时，选中代码段后需点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fa/v3/qwlWuK0zTrqeJFIToxkJIQ/zh-cn_image_0000002561832619.png?HW-CC-KV=V1&HW-CC-Date=20260429T054439Z&HW-CC-Expire=86400&HW-CC-Sign=BC321C29F705F4BE48A8CC12CA7041222E821F54DA2ED240C9F8AB5B74DF9693)图标开启光标上下文功能。

在DevEco Studio 6.0.2 Beta1版本，分析代码文件时，支持在对话框输入要分析的代码文件或直接分析当前文件；分析代码片段时，选中代码段后直接分析，无需开启![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7b/v3/jjp5yAguSfKwFj0J1tqTjg/zh-cn_image_0000002561832615.png?HW-CC-KV=V1&HW-CC-Date=20260429T054439Z&HW-CC-Expire=86400&HW-CC-Sign=FE52DA9BC69B46A98AD7AA13DF99B6C79B9E8AA076DC696E1E28B7C88FF649F8)图标。

从DevEco Studio 6.0.2 Release版本开始，使用HarmonyOS Ask智能体分析代码文件。

以DevEco Studio 6.1.0 Release和DevEco Studio 6.0.1 Beta1版本为例说明，操作如下。

**DevEco Studio 6.1.0 Release版本**

1. 选择HarmonyOS Ask智能体，在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，选择需要分析的代码文件，或在对话框输入文件路径指定需要分析的代码文件，或分析当前代码文件。
2. 在对话框输入描述，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9b/v3/jbasNa6cR_O0lNlyTpnGRQ/zh-cn_image_0000002561752635.png?HW-CC-KV=V1&HW-CC-Date=20260429T054439Z&HW-CC-Expire=86400&HW-CC-Sign=1648F4CE507510DC43B214FEF5623A5C1D3B06F391AA809D5AEBAFF6E8E41D4A)发送后等待回复。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/51/v3/Y9Vp6lpxTnCmQJHJ5FySrA/zh-cn_image_0000002530752696.gif?HW-CC-KV=V1&HW-CC-Date=20260429T054439Z&HW-CC-Expire=86400&HW-CC-Sign=9F0EE92BBEF5F1628757CBE96143FC6E37E3F702855263310DEAF637A3481A36)

**DevEco Studio 6.0.1 Beta1版本**

分析代码文件：在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，可对单个或多个代码文件进行分析。

分析代码片段：选中代码段，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ae/v3/oTxCL2TLSX-G0AU6cOKZ3g/zh-cn_image_0000002530912694.png?HW-CC-KV=V1&HW-CC-Date=20260429T054439Z&HW-CC-Expire=86400&HW-CC-Sign=F5000FE834F0F343A956FD11F60B0271DA7688A0DEBA0621610B7D0F215E5D6A)图标开启光标上下文功能，在对话框内描述，开始分析。
