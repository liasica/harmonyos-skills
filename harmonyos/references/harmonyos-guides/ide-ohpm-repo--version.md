查询ohpm-repo版本。

## 命令格式

```
1. ohpm-repo --version
2. alias: -v
```

## 功能描述

打印ohpm-repo的版本号。

## 示例

执行以下命令，查看版本信息：

```
1. ohpm-repo -v
```
