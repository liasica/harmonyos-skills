## 函数功能

获取解析算子属性的函数。

## 函数原型

```
1. ParseParamFunc GetParseParamFn() const
```

## 参数说明

无

## 约束说明

GetParseParamFn接口后续版本将会废弃，请使用[ParseParamsByOperatorFn](cannkit-parseparamsbyoperatorfn.md)接口获取算子属性。
