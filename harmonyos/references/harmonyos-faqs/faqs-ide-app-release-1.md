**问题现象**

打开上传软件包界面后，提示”No app with the specified package name is found under the team.”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fc/v3/5zX3DLfdTKmghsehs1KMZQ/zh-cn_image_0000002541844403.png?HW-CC-KV=V1&HW-CC-Date=20260429T062137Z&HW-CC-Expire=86400&HW-CC-Sign=DAD3E892BC195F74ABAA7D2CBCEE20F8585EEEB4F8514D2AD51F7A8D92AAC1E1)

**解决措施**

因为当前账号在AppGallery Connect没有对应包名的应用，请在AppGallery Connect登录当前账号后[创建应用](../app/agc-help-createapp-0000001146718717.md)，再重新打开上传软件包界面。
