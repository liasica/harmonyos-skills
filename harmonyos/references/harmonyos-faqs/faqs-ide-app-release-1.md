**问题现象**

打开上传软件包界面后，提示”No app with the specified package name is found under the team.”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fc/v3/5zX3DLfdTKmghsehs1KMZQ/zh-cn_image_0000002541844403.png?HW-CC-KV=V1&HW-CC-Date=20260428T003019Z&HW-CC-Expire=86400&HW-CC-Sign=CA623BCE2BD59A7EE51A75ED15C08D482A64AEE166544A51C6573EE389C8BCFC)

**解决措施**

因为当前账号在AppGallery Connect没有对应包名的应用，请在AppGallery Connect登录当前账号后[创建应用](../app/agc-help-createapp-0000001146718717.md)，再重新打开上传软件包界面。
