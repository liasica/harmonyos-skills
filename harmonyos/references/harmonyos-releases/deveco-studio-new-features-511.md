* 支持对C API的变更情况进行查询，提供跨版本查询能力。具体请参考[API变更查询](../harmonyos-guides/ide-editer-overview.md#section234174215231)。
* hvigor-config.json5文件的execution下新增optimizationStrategy字段，用于指定构建模式。同时命令行参数支持--optimization-strategy=performance/memory。具体请参考[hvigor-config.json5文件](../harmonyos-guides/ide-hvigor-set-options.md)和[hvigorw](../harmonyos-guides/ide-hvigor-commandline.md)。
* 新建Native C++工程默认使用毕昇编译器，打开历史工程会弹窗提示，点击**立即体验**可以切换使用毕昇编译器。

## DevEco Studio 5.1.1 Beta1（5.1.1.420）新增和增强特性

* DevEco Studio支持开发API 19工程。
* 工程级、模块级build-profile.json5新增以下字段。具体请参考[工程级build-profile.json5文件](../harmonyos-guides/ide-hvigor-build-profile-app.md)和[模块级build-profile.json5文件](../harmonyos-guides/ide-hvigor-build-profile.md)。
  + buildOption下新增generateSharedTgz，用于指定编译HSP模块时是否生成tgz包。
  + buildOption/resOptions下新增ignoreResourcePattern，用于对资源目录resources或开发者自定义的资源目录下的文件/文件夹名称进行过滤，匹配到的文件不会被打包到产物中。
* hvigor-config.json5的properties下新增以下字段。具体请参考[hvigor-config.json5文件](../harmonyos-guides/ide-hvigor-set-options.md)。
  + 新增ohos.byteCodeHar.integratedOptimization，用于指定是否开启字节码HAR集成行为优化。
  + 新增hvigor.incremental.optimization，用于指定是否开启任务增量判断优化。
  + 新增hvigor.task.schedule.optimization，用于指定是否开启任务调度优化。
* Build Analyzer支持查看构建过程的内存消耗情况。具体请参考[查看构建过程内存消耗曲线图](../harmonyos-guides/ide-hvigor-build-analyzer.md#section71431248981)。
* DevEco Studio支持使用设备连接助手排查设备连接后无法识别的问题。具体请参考[使用设备连接助手排查问题](../harmonyos-guides/ide-run-device.md#section155318499334)。
* Hot Reload能力增强，支持热重启。具体请参考[Hot Reload](../harmonyos-guides/ide-hot-reload.md)。
* ArkTS代码调试支持查看异步函数堆栈。具体请参考[查看异步函数堆栈](../harmonyos-guides/ide-debug-async-stack-traces.md)。
* Instrument Test支持通过命令行启用ASan检测。具体请参考[使用命令行执行测试Instrument Test](../harmonyos-guides/ide-instrument-test.md#section14255191913322)。
